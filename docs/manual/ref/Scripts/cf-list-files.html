cf-list-files

Purpose

Displays a directory listing.

Description

Lists files in the specified directory on the server.

Usage: 

grails cf-list-files [path] [--instance] [--appname]

Required arguments:

  • path - The relative path to the file on the server; defaults to the root

Optional arguments:

  • appname - The name of the application
  • instance - The application instance number; defaults to 1
