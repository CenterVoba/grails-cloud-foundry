cf-stats

Purpose

Report resource usage for an application.

Description

Displays information about resource usage for your application. Displays information for all instances, including:

  • CPU usage
  • number of CPU cores
  • current memory usage
  • total memory
  • current disk usage
  • total disk
  • uptime

Usage: 

grails cf-stats [--appname]

Required arguments: none .

Optional arguments:

  • appname - The name of the application
