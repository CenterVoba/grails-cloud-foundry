cf-delete-all-apps

Purpose

Deletes all applications.

Description

Deletes all registered applications. Will also delete bound services if the force argument is specified or if you answer Y to the delete prompt. Don't delete any service that will be used again or which is currently in use by another application.

Usage: 

grails cf-delete-all-apps [--force]

Required arguments: none .

Optional arguments:

  • force - If specified then all services will be deleted without asking
